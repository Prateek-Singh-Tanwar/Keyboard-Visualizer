# Keyboard Visualizer

A simple **interactive keyboard visualizer** built with HTML, CSS, and JavaScript. Press any key on your physical keyboard, and the corresponding key will highlight on the visual keyboard. The last pressed key is displayed prominently above the keyboard.

## Demo

You can try the visualizer by opening the `index.html` file in your browser.

## Features

- Highlights the key pressed on the on-screen keyboard.
- Displays the last key pressed in a large box.
- Supports common keys like letters, numbers, symbols, and modifier keys (`Shift`, `Ctrl`, `Alt`, `CapsLock`, etc.).
- Smooth animation and responsive design.
