# Keyboard Visualizer

A **real-time interactive keyboard visualizer** built with HTML, CSS, and JavaScript.  

This project helps you **practice correct finger positioning** on the keyboard, which is essential for increasing typing speed and accuracy. Each key is color-coded to indicate which finger should press it, and pressing a key highlights it both on the visual keyboard and in the key display above.

**Live Demo:** [Try it here](https://prateek-singh-tanwar.github.io/Keyboard-Visualizer/)

---

## Motivation / Backstory

I created this visualizer to **improve my typing speed**. Before typing fast, you need to ensure your **hands and fingers are positioned correctly**.  

This visualizer assigns **different colors to keys based on finger placement**, so you can learn proper finger positioning in a visual and interactive way.  

---

## Finger Positions Based on Key Colors

### 1. Pink Keys
**Fingers:** Left pinky & Right pinky  
**Keys:**  
- Left pinky: `~`, `1`, `Tab`, `CapsLock`, `Shift`, `Ctrl`  
- Right pinky: `Backspace`, `Enter`, `Shift`, `Ctrl`, `\`, `]`, `[`, `-`, `=`

### 2. Yellow Keys
**Fingers:** Left ring finger & Right ring finger  
**Keys:**  
- Left ring: `2`, `W`, `S`, `X`, `Alt` (left side)  
- Right ring: `8`, `I`, `K`, `,`, `Alt` (right side)

### 3. Green Keys
**Fingers:** Left middle finger & Right middle finger  
**Keys:**  
- Left middle: `3`, `E`, `D`, `C`  
- Right middle: `7`, `U`, `J`, `N`

### 4. Red Keys
**Fingers:** Left index finger & Right index finger  
**Keys:**  
- Left index: `4`, `5`, `R`, `T`, `F`, `G`, `V`, `B`  
- Right index: `6`, `Y`, `H`, `J`, `M`

### 5. Blue Keys
**Fingers:** Left & Right middle fingers for numbers and top row  
**Keys:**  
- Left hand: `6`  
- Right hand: `Y`, `U`, `H`, `J`

### 6. Purple Keys
**Fingers:** Right pinky for punctuation and special keys on the right side  
**Keys:**  
`P`, `{`, `}`, `:`, `'`, `/`, `Shift`, `Ctrl`

---

### Summary of Finger Home Row Positions

- **Left hand fingers rest on:**  
`A` (pinky), `S` (ring), `D` (middle), `F` (index)  

- **Right hand fingers rest on:**  
`J` (index), `K` (middle), `L` (ring), `;` (pinky)

---

## Features

- Highlights pressed keys on a virtual on-screen keyboard  
- Displays the last key pressed in a large key box  
- Smooth animations for key press effects  
- Supports letters, numbers, symbols, and modifier keys (`Shift`, `Ctrl`, `Alt`, `CapsLock`)  
- Prevents unwanted scrolling when pressing keys like `Space` or arrow keys  
- Fully responsive design for different screen sizes  
- Helps visualize proper finger placement for faster typing
